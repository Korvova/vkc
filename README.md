# VKC

Electron-приложение для переговорной комнаты с отдельным окном захвата HDMI-входа ноутбука и запуском веб-ВКС на мини-ПК.

## Что делает софт

- Открывает веб-страницу ВКС в основном полноэкранном окне.
- Принудительно использует основную USB-камеру переговорки как камеру звонка.
- Поднимает отдельное скрытое окно `Мой рабочий стол`.
- В окне `Мой рабочий стол` показывает сигнал с HDMI-to-USB карты захвата, к которой подключается ноутбук.
- Позволяет в ВКС расшарить окно `Мой рабочий стол`, чтобы показать экран ноутбука участникам звонка.
- Оставляет обычную камеру активной для видео во время демонстрации.
- Показывает маленькую панель управления поверх интерфейса ВКС.
- Дает кнопки `Назад к расписанию` и `Закрыть`.

## Как это работает

### Основные окна

- Главное окно открывает страницу ВКС или внутреннюю страницу переговорки.
- Скрытое окно `Мой рабочий стол` постоянно захватывает HDMI-вход через карту захвата.
- Панель управления поверх основного окна позволяет вернуться на стартовую страницу или закрыть приложение.

### Камера и демонстрация

- Для обычного видеозвонка приложение старается выбрать основную камеру переговорки `A4tech FHD 1080P PC Camera`.
- Для окна `Мой рабочий стол` приложение использует HDMI-захват `USB3.0 Capture Video`.
- При нажатии демонстрации экрана приложение сохраняет камеру звонка и отдает в демонстрацию поток с ноутбука.

## Сценарий использования

1. На мини-ПК запускается приложение.
2. На основном экране открывается нужная страница ВКС.
3. Пользователь подключает ноутбук по HDMI через USB-карту захвата.
4. ВКС использует основную переговорную камеру для видео.
5. При демонстрации экрана пользователю доступен поток ноутбука через окно `Мой рабочий стол`.

## Файлы проекта

- `main.js` - основная логика Electron, окна, разрешения, выбор камеры и демонстрации.
- `capture.html` - окно предпросмотра HDMI-захвата ноутбука.
- `main-preload.js` - мост между окном ВКС и main-процессом.
- `control.html` - маленькая панель управления.
- `control-preload.js` - IPC для кнопок панели управления.

## Запуск

```bash
npm install
npm start
```

## Что нужно для работы

- Windows-ПК или мини-ПК.
- USB-вебкамера переговорки.
- HDMI-to-USB карта захвата.
- Ноутбук, подключаемый к карте захвата по HDMI.

## Текущий статус

- Камера звонка и демонстрация ноутбука работают раздельно.
- Окно `Мой рабочий стол` по умолчанию скрыто.
- Панель управления позволяет закрыть приложение без `Alt+F4`.
